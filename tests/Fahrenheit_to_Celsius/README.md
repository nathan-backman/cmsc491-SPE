# Fahrenheit to Celsius Conversion Test
A simple test that takes in integers as Fahrenheit temperatures, performs a 
conversion on them, passes them to an operator that formats them as "Celsius 
Temperature: *number*", which then passes the number to an output operator. 

### Operator Workflow
 1) Change integer from Fahrenheit to Celsius.
 2) Transform the data to be easily readable.
 3) Output the data.

### Input data format
Format: ```integer```
  * **integer:** Simple integer input as strings

### Output format
Format: ```Celsius Temperature: [integer]```
  * **integer:** The Celsius version of the temperature in integer form
